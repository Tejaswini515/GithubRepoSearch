# GithubRepoSearcg
React App to search repositories of github users with username. Created with create-react-app.

## Usage
In a terminal run:

```
git clone https://github.com/Tejaswini515/GithubRepoSearch.git
cd GithubRepoSearch
npm install
npm start
```
